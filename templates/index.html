RUL Prediction Using LSTM

This project aims to predict the Remaining Useful Life (RUL) of censored filter test cycles. Training and test data are provided, each containing 35 and 20 run-to-failure cycles, respectively. The test data includes randomly right-censored run-to-failure trajectories and their RUL for prediction. The challenge is to predict RUL accurately despite varying data qualities.

Detailed setup descriptions and various physical filter models in literature support the data-driven models by integrating physical knowledge, leading to theory-guided data science or informed machine learning.

- Test bench records flow rate, particle feed, and differential pressure across the filter during a lifetime test.
- Flow sensor location upstream of particle feed causes falsification, compensated by a modeled function.
- Differential pressure sensor has a 0 to 2500 Pa measuring range.
- Sensor connections in filter chamber designed to minimize interference.
- Data in dataset represents corrected values.
- Operating modes selectable via valves for flow or pressure control.
- All life tests in dataset conducted under flow control.
- Standardized procedure for measurements with consistent starting positions and stop criteria.
- Parameters set in user interface before each measurement.
